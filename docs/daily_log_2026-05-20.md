## What We Covered

- **IMDb Data Pipeline Debugging** — Cloud Composer environment has zombie tasks from BashOperator timeout. Architecture recommendation: run extracts locally (Mac, 2s/request), upload to GCS, let Composer handle only sensor → BQ load.

Fix: add explicit TERMINAL_ENV exception. Lesson: competing PR #29239 fixed at the chokepoint (env_loader.py) vs our 2-call-site approach.

- **Composer zombie tasks**: BashOperator task hangs >120s → Celery kills it → zombie remains. Root cause: extract script's HTTP call to RapidAPI has no timeout. Fix: `timeout 120` bash wrapper + `execution_timeout=timedelta(minutes=5)`. Architecture insight: Composer workers can't reliably reach external APIs — run network-bound work locally.
