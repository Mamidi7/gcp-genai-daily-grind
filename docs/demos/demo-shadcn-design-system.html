Cards

Four cards in a grid. Hover for lift effect. Click to trigger action.

Standard

Gunicorn

Production WSGI server for Flask

Manages worker processes, handles concurrent requests. Use exec-form CMD in Dockerfile.
Docker

Layer Caching

COPY order matters

Copy requirements.txt first, run pip install, then copy app code. Avoids reinstalling deps on every build.
Gotcha

Port Binding

0.0.0.0 not 127.0.0.1

Inside a container, 127.0.0.1 binds to container localhost only. Cloud Run's health check won't reach it. Always use 0.0.0.0.

Tabs System

Click tabs to switch content. shadcn-style tab bar.

Flask

Flask is a lightweight WSGI web application framework. It's designed to make getting started quick and easy, with the ability to scale up to complex applications. On Cloud Run, Flask handles route dispatching, request parsing, and response generation.

Gunicorn

Gunicorn 'Green Unicorn' is a Python WSGI HTTP Server for UNIX. It's the production-grade server that sits between Cloud Run and Flask. Manages worker processes, handles concurrent connections, and forwards requests to Flask via the WSGI protocol.

Cloud Run

Cloud Run is a managed compute platform that lets you run containers directly on Google's infrastructure. It auto-scales from 0 to N instances, charges per request, and handles HTTPS, auth, and monitoring out of the box. No Kubernetes knowledge needed.
