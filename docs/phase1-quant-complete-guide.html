01

Data Interpretation — 8-10 Marks

The single biggest section. Master 4 chart types.
DI Speed Rules

Never do full calculation in DI. Estimate first.

  1. Round aggressively: 247 → 250, 1983 → 2000. Compare, don't compute exact.
  2. Use fractions: 25% = 1/4, 33% = 1/3, 50% = 1/2, 66% = 2/3, 75% = 3/4.
  3. For % increase: (New - Old)/Old. If New = 120, Old = 100 → increase = 20/100 = 20%.
  4. For ratio: Simplify by dividing both by 10, 100, or common factor. 450:600 = 45:60 = 3:4.
  5. Average of A and B: (A+B)/2. If A=240, B=260 → avg=250 (midpoint).
GOLDEN RULE: Read the QUESTION first, then look at chart. Don't study the chart. 80% of chart data is irrelevant to any single question.

A: (250-200)/200 = 50/200 = 25% B: (360-300)/300 = 60/300 = 20% C: (180-150)/150 = 30/150 = 20% D: DECREASE (skip) E: (300-250)/250 = 50/250 = 20%

Q2: Total sales in 2024 is what % of total sales in 2023?

A = 25% of 2400 = 600 C = 15% of 2400 = 360 Difference = 600 - 360 = 240 D = 30% of 2400 = 720 % = 240/720 × 100 = 1/3 × 100 = 33.33%
Answer: 33.33% or 33(1/3)%
Shortcut: Since total is same for all, use % directly: (25-15)/30 = 10/30 = 33.33%. Total value (2400) cancels out!

Number Series — 3-5 Marks

Wrong number series + missing number. 8 patterns cover 95%.
Pattern 1: +2, +4, +6, +8...
2, 4, 8, 14, 22, 32 (+2,+4,+6,+8,+10)
Pattern 2: ×2+1, ×2+2, ×2+3...
1, 3, 7, 15, 31 (×2+1 repeated)
Pattern 3: Squares ± n
2, 5, 10, 17, 26 (1²+1, 2²+1, 3²+1...)
Pattern 4: Cubes ± n
7, 26, 63, 124 (2³-1, 3³-1, 4³-1, 5³-1)
Pattern 5: Alternate
1, 2, 4, 8, 16 (+1,+2,+4,+8) OR ×2
Pattern 6: n²-n
0, 2, 6, 12, 20 (0×1, 1×2, 2×3, 3×4...)
Pattern 7: Prime numbers
2, 3, 5, 7, 11, 13, 17...
Pattern 8: Double step
+3, ×2, +3, ×2, +3, ×2...
The 30-Second Method

Check patterns in this order

  1. Step 1: Calculate differences between consecutive numbers.
  2. Step 2: If differences have a pattern (+2,+4,+6) → solved.
  3. Step 3: If differences are same → arithmetic progression.
  4. Step 4: If no pattern in differences, check ratios (×2, ×3).
  5. Step 5: If ratios don't work, check if numbers are near squares/cubes (25,36,49 or 27,64,125).
  6. Step 6: If still stuck, check alternate positions (1st, 3rd, 5th vs 2nd, 4th, 6th).
TRICK: For wrong number series, find the pattern first, then find which number breaks it. DON'T check each number one by one.
Example — Wrong Number Series
Find the wrong number: 3, 7, 15, 31, 61, 127, 255
Pattern check: 3×2+1=7, 7×2+1=15, 15×2+1=31, 31×2+1=63, 63×2+1=127 But series has 61 where 63 should be
Answer: 61 is wrong (should be 63)
Example — Missing Number
4, 12, 36, 108, ?, 972
Ratios: 12/4=3, 36/12=3, 108/36=3 Geometric progression with common ratio 3 Missing = 108 × 3 = 324
Answer: 324
Example — Square Pattern
5, 11, 21, 37, 61, ?
Differences: +6, +10, +16, +24 Second differences: +4, +6, +8 Next second diff = +10, so next first diff = 24+10 = 34 Or: 2²+1=5, 3²+2=11, 4²+5=21... actually check: 2²+1, 3²+2, 4²+5... no pattern Better: n²+(n-1): 2²+1=5, 3²+2=11, 4²+5? No. Try n²+(n+1): no. Simplest: 5=1×5, 11=2×5+1, 21=4×5+1, 37=7×5+2... messy. Actually: 5=2²+1, 11=3²+2, 21=4²+5, 37=6²+1? No. Let's try 2n²+something. 2(1)²+3=5, 2(2)²+3=11, 2(3)²+3=21, 2(4)²+5=37? Inconsistent. Try: 1×2+3=5, 2×3+5=11, 3×4+9=21, 4×5+17=37... differences in added terms: +2,+4,+8 (×2). Next = +16. 5×6+33=63? Let's check: 61+? Actually 5×6+17=47? No. Better approach: 5, 11, 21, 37, 61. Differences: 6, 10, 16, 24. These are n²+n: 2²+2=6, 3²+1=10? No. 1×6=6, 2×5=10, 4×4=16, 6×4=24? No. Actually 6=2×3, 10=2×5, 16=2×8, 24=2×12. Multipliers: 3,5,8,12. Diffs: +2,+3,+4. Next diff = +5, so next multiplier = 17. Next diff = 2×17 = 34. So 61+34 = 95.
Answer: 95 (or check if there's a cleaner pattern: n³-n²+1: 1-1+1=1 no. 2n²-n+? no. The diff method is reliable: 95)

Simplification & Approximation

3-5 questions. BODMAS + fraction tricks. 30 seconds each.
Fraction-Decimal Conversions (Memorize)
Must-Know Equivalents
1/2 = 0.5 | 1/3 ≈ 0.333 | 1/4 = 0.25 | 1/5 = 0.2 | 1/6 ≈ 0.167
1/8 = 0.125 | 1/9 ≈ 0.111 | 1/10 = 0.1 | 1/12 ≈ 0.083 | 1/16 = 0.0625
1/20 = 0.05 | 1/25 = 0.04 | 1/50 = 0.02
For multiples: 3/8 = 3×0.125 = 0.375. 5/8 = 5×0.125 = 0.625. 7/8 = 0.875.
BODMAS + Shortcut Rules

Order: Brackets, Orders (powers), Division, Multiplication, Addition, Subtraction

  1. a + (b - c) = a + b - c — Bracket with +, just remove.
  2. a - (b - c) = a - b + c — Bracket with -, flip signs inside.
  3. (a+b)² - (a-b)² = 4ab — Saves 30 seconds.
  4. a² - b² = (a+b)(a-b) — Difference of squares.
  5. Multiplication by 5: Divide by 2, then ×10. 64×5 = 32×10 = 320.
  6. Multiplication by 25: Divide by 4, then ×100. 48×25 = 12×100 = 1200.
  7. Multiplication by 125: Divide by 8, then ×1000. 64×125 = 8×1000 = 8000.
TRICK: 5=10/2, 25=100/4, 125=1000/8. These come up constantly in simplification. Practice these daily.
Example — Simplification
45% of 240 + 35% of 360 - 25% of 480 = ?
45% of 240 = (45×240)/100 = 45×2.4 = 108 35% of 360 = (35×360)/100 = 35×3.6 = 126 25% of 480 = 480/4 = 120 108 + 126 - 120 = 234 - 120 = 114
Answer: 114
Shortcut: 45% = 50%-5%. 50% of 240=120, 5% of 240=12, so 45%=108. 35%=30%+5%. 30% of 360=108, 5%=18, so 35%=126.
Example — Approximation
(24.87 × 15.12) / 7.98 ≈ ?
Round: 24.87 ≈ 25, 15.12 ≈ 15, 7.98 ≈ 8 25 × 15 = 375 375 / 8 ≈ 46.875 ≈ 47
Answer: ~47 (exact: 47.09)
Example — Surds
If √? + 24 = 38, then ? =
√? = 38 - 24 = 14 ? = 14² = 196
Answer: 196

Quadratic Equations — 3-5 Marks

Two equations, compare roots x and y. Quick factorization trick.
Quadratic Formula
ax² + bx + c = 0
x = [-b ± √(b² - 4ac)] / 2a
But factoring is faster. Product of roots = c/a. Sum = -b/a.
The Factorization Shortcut

Find two numbers that multiply to (a×c) and add to b

  1. Example: x² + 7x + 12 = 0. Find two numbers that multiply to 12 and add to 7.
  2. 3 × 4 = 12, 3 + 4 = 7. So (x+3)(x+4) = 0. Roots: -3, -4.
  3. Example: x² - 5x + 6 = 0. Multiply to 6, add to -5.
  4. -2 × -3 = 6, -2 + (-3) = -5. So (x-2)(x-3) = 0. Roots: 2, 3.
  5. Example: x² + 5x - 6 = 0. Multiply to -6, add to 5.
  6. 6 × (-1) = -6, 6 + (-1) = 5. So (x+6)(x-1) = 0. Roots: -6, 1.
TRICK: Always write "split middle term" mentally: x² + 7x + 12 = x² + 3x + 4x + 12 = x(x+3) + 4(x+3) = (x+3)(x+4).
Example — Compare x and y
I. x² - 9x + 20 = 0
II. y² - 13y + 42 = 0
I: Multiply to 20, add to -9 → -4, -5. (x-4)(x-5)=0. x = 4, 5. II: Multiply to 42, add to -13 → -6, -7. (y-6)(y-7)=0. y = 6, 7. Compare: x max = 5, y min = 6. So x < y ALWAYS.
Answer: x < y
Example — No Clear Relation
I. x² - 5x + 6 = 0
II. y² - 7y + 10 = 0
I: x = 2, 3 (factors: -2,-3) II: y = 2, 5 (factors: -2,-5) Compare: If x=3 and y=2, x>y. If x=2 and y=5, x
Answer: No relation (x = y or relation cannot be established)
Common Trap: When both x and y have same values (x=2,3 and y=2,5), answer is "No relation" NOT "x ≤ y". Only if ALL x values are less than ALL y values, then x < y.

Profit & Loss

1-2 questions. CA students know this. Just speed up.
Profit %
(SP - CP)/CP × 100
SP = CP × (100+P%)/100
Loss %
(CP - SP)/CP × 100
SP = CP × (100-L%)/100
Marked Price
MP = CP × (100+Markup%)/100
Discount = MP - SP
Equivalent Discount
Successive: a% then b% = a+b-ab/100
20%+20% = 20+20-4 = 36%
Dishonest Dealer
Cheats by x% in weight, gains y%
True profit = [(100+y)/(100-x) - 1] × 100
CP from SP & P%
CP = SP × 100/(100+P%)
If SP=132, P%=10%: CP=132×100/110=120
Example — Dishonest Dealer
A shopkeeper marks goods 40% above CP and allows 20% discount. What is profit %?
Let CP = 100 MP = 100 + 40% = 140 SP after 20% discount = 140 × 0.8 = 112 Profit = 112 - 100 = 12 Profit % = 12/100 × 100 = 12%
Answer: 12%
Shortcut: Net effect formula: +40% then -20% = 40 - 20 - (40×20)/100 = 20 - 8 = +12%. Same answer, 10 seconds.

Time, Speed & Distance

1-2 questions. Relative speed + train problems.
Basic
Speed = Distance/Time
Time = Distance/Speed
Distance = Speed × Time
Average Speed
Same distance: 2ab/(a+b)
At 40 km/h and 60 km/h: avg = 2×40×60/100 = 48
Relative Speed
Same direction: a - b
Opposite: a + b
Train crossing
Platform: (L_train + L_platform)/Speed
Pole/man: L_train/Speed
Boat & Stream
Downstream: u+v
Upstream: u-v
u = (down+up)/2, v = (down-up)/2
Circular Track
Meeting: Relative speed × time = circumference
Same direction: faster laps slower
Example — Train Crossing
A 200m train at 72 km/h crosses a platform in 18 sec. What is platform length?
Speed = 72 km/h = 72 × 5/18 = 20 m/s Total distance = Speed × Time = 20 × 18 = 360m Total distance = Train length + Platform length 360 = 200 + Platform → Platform = 160m
Answer: 160m
Example — Average Speed
A man travels from A to B at 40 km/h and returns at 60 km/h. Find average speed.
Same distance formula: 2ab/(a+b) = 2×40×60/100 = 4800/100 = 48
Answer: 48 km/h
NEVER average the speeds: (40+60)/2 = 50 is WRONG. Always use harmonic mean for same distance.

Work & Time + Pipes & Cisterns

1-2 questions. Efficiency method is fastest.
Basic Work
A does work in 10 days → 1/10 per day
B does work in 15 days → 1/15 per day
A+B = 1/10 + 1/15 = 1/6 → 6 days
LCM Method
LCM of 10,15 = 30 units
A = 3 units/day, B = 2 units/day
A+B = 5 units/day → 30/5 = 6 days
Pipes: Filling
Pipe A fills in 4 hrs → 1/4 tank/hr
Pipe B fills in 6 hrs → 1/6 tank/hr
Both: 1/4+1/6 = 5/12 → 12/5 = 2.4 hrs
Pipes: Empty
Filling pipe + emptying pipe: SUBTRACT
A fills 1/4, B empties 1/6 → Net = 1/4-1/6 = 1/12 → 12 hrs
Work with wages
Ratio of wages = Ratio of work done
Or = Ratio of efficiency × days
Example — Work with Leave
A can do work in 12 days, B in 18 days. They work together but B leaves 3 days before completion. Total days?
LCM of 12,18 = 36 units A = 3 units/day, B = 2 units/day Last 3 days only A works: 3 × 3 = 9 units Remaining work = 36 - 9 = 27 units A+B together = 5 units/day → 27/5 = 5.4 days Total = 5.4 + 3 = 8.4 days
Answer: 8.4 days (or 42/5 days)

Ratio, Average & Alligation

1-2 questions each. Alligation is the fastest method for mixtures.
Alligation (Mixture Method)
If two groups with averages a and b mix to give average A
Ratio of quantities = (b - A) : (A - a)
Where a < A < b. Subtract cross-wise. Simple.
Example — Alligation
A class has boys avg weight 50kg, girls avg 40kg. Class avg is 46kg. Find boy:girl ratio.
Boys: 50, Girls: 40, Average: 46 Boys quantity / Girls quantity = (46-40) : (50-46) = 6:4 = 3:2
Answer: 3:2
Visual: Write 40 and 50. 46 in middle. Differences: 6 and 4. Ratio = 6:4 = 3:2. Done in 10 seconds.
Example — Average with Replacement
Average of 11 numbers is 45. If one number (36) is replaced by 56, what is new average?
Total of 11 numbers = 11 × 45 = 495 New total = 495 - 36 + 56 = 515 New average = 515/11 = 46.81 ≈ 46.8
Answer: 46.8 (or 515/11)
Shortcut: Change in one number = +20. Spread over 11 numbers = +20/11 ≈ +1.8. New avg = 45 + 1.8 = 46.8. No need to calculate total!

Mensuration

1-2 questions. RBI asks simple 2D shapes. π = 22/7 usually.
Rectangle
Area = L × B
Perimeter = 2(L+B)
Diagonal = √(L²+B²)
Square
Area = a²
Perimeter = 4a
Diagonal = a√2
Circle
Area = πr²
Circumference = 2πr
If diameter = d, area = πd²/4
Triangle
Area = ½ × base × height
Equilateral: Area = (√3/4)a²
Heron: √[s(s-a)(s-b)(s-c)]
Cylinder
Volume = πr²h
CSA = 2πrh
TSA = 2πr(r+h)
Sphere
Volume = (4/3)πr³
Surface = 4πr²
Example — Area Change
If radius of a circle is increased by 20%, by what % does area increase?
New radius = 1.2r New area = π(1.2r)² = 1.44πr² Increase = 1.44 - 1 = 0.44 = 44%
Answer: 44%
Shortcut: If linear dimension increases by x%, area increases by (2x + x²/100)%. Here 20%: 40 + 4 = 44%. If 3D: 3x + 3x²/100 + x³/10000.

Ages, Partnership, Interest

0-1 question each. CA-level. Just formula recall.
Ages
n years ago: subtract n from both
n years hence: add n to both
Ratio remains same when added to both: a:b = (a+x):(b+x) only if a=b
Partnership
Profit ratio = (Capital × Time)
A invests 5000 for 12 mo, B invests 8000 for 6 mo
Ratio = 5000×12 : 8000×6 = 60000:48000 = 5:4
Simple Interest
SI = P×R×T/100
Amount = P + SI
If SI = P in T years, R = 100/T %
Compound Interest
A = P(1+R/100)^T
For 2 years: CI = P[R²/10000 + 2R/100]
Difference CI-SI for 2 years = P(R/100)²
Example — Ages
Father is 3 times as old as son. After 15 years, father will be twice as old as son. Present age of son?
Let son = x, father = 3x After 15 years: 3x+15 = 2(x+15) 3x + 15 = 2x + 30 x = 15
Answer: Son = 15 years
