01

Simplification — Free Marks

5-6 questions. Should take 15 seconds each. BODMAS + fraction tricks.
BODMAS Reminder

Order of operations — never get it wrong

  1. Brackets (solve innermost first)
  2. Order (powers, roots)
  3. Division and Multiplication (left to right)
  4. Addition and Subtraction (left to right)
SHORTCUT: In exam questions, brackets and orders are usually straightforward. Most mistakes happen at DM vs AS. Always do division/multiplication BEFORE addition/subtraction.
Fraction Trick

Convert everything to same denominator or decimals quickly

  1. 1/2 = 0.5, 1/4 = 0.25, 1/5 = 0.2, 1/8 = 0.125, 1/10 = 0.1
  2. 1/3 = 0.333, 2/3 = 0.666, 1/6 = 0.166, 1/9 = 0.111
  3. If question has 3/4 + 5/8 - 1/2, convert: 0.75 + 0.625 - 0.5 = 0.875
MEMORIZE: 1/2, 1/3, 1/4, 1/5, 1/6, 1/8, 1/9, 1/10 as decimals. Saves 10 seconds per question.
Example
35% of 280 + ? = 154
35% of 280 = 0.35 x 280 = 98 98 + ? = 154 ? = 154 - 98 = 56
Answer: 56
Example — Approximation
152.4 x 3.96 + 18.2 = ?
Approximate: 152 x 4 = 608 608 + 18 = 626 Exact would be slightly less (since 3.96 < 4), but closest option = 626
Answer: ~626 (pick closest option)

03

Number Series

3-5 questions. 6 pattern types cover 90% of exam questions.
1. Addition/Subtraction Series
2, 5, 8, 11, 14... (adding 3 each time)
20, 17, 14, 11... (subtracting 3)
2. Multiplication/Division Series
2, 6, 18, 54... (x3 each time)
96, 48, 24, 12... (/2 each time)
3. Square/Cube Series
1, 4, 9, 16, 25... (1², 2², 3²...)
1, 8, 27, 64... (1³, 2³, 3³...)
4. Alternate Series
2, 5, 4, 7, 6, 9... (Two series merged: 2,4,6... and 5,7,9...)
5. Difference of Differences
2, 3, 5, 8, 12...
Diff: +1, +2, +3, +4...
Next diff = +5, so next = 17
6. Prime Number Series
2, 3, 5, 7, 11, 13, 17... (prime numbers)
The 10-Second Method

How to crack any number series in 10 seconds

  1. Step 1: Calculate first differences (subtract each from next)
  2. Step 2: If differences are same = simple add/subtract
  3. Step 3: If differences change by same amount = difference of differences
  4. Step 4: If differences are multiplying = multiplication series
  5. Step 5: If numbers look familiar = check squares/cubes/primes
  6. Step 6: If nothing works = alternate series (split odd and even positions)
SHORTCUT: Always check if numbers are near squares (15²=225, 16²=256) or cubes. If 125 appears → think 5³. If 343 appears → think 7³. Exam series LOVE squares and cubes.
Example — Difference of Differences
3, 8, 17, 30, 47, ?
Differences: 5, 9, 13, 17... Diff of diffs: +4, +4, +4... Next diff = 17 + 4 = 21 Next number = 47 + 21 = 68
Answer: 68
Example — Alternate Series
12, 10, 14, 8, 18, 6, ?
Odd positions (1,3,5,7): 12, 14, 18, ? → +2, +4, +6... Even positions (2,4,6): 10, 8, 6 → -2 each 7th position = odd → 18 + 6 = 24
Answer: 24
04

Quadratic Equations — Free Marks

3-5 questions. Should take 20 seconds each. Two methods.
Method 1: Factorization

For equations where factors are easy to spot

  1. x² + 7x + 12 = 0 → Find two numbers that multiply to 12 and add to 7 → 3 and 4
  2. (x + 3)(x + 4) = 0 → x = -3 or -4
  3. x² - 5x + 6 = 0 → Numbers: -2 and -3 (multiply to 6, add to -5)
  4. (x - 2)(x - 3) = 0 → x = 2 or 3
TRICK: If middle term is + and last term is +, both factors are positive. If middle is - and last is +, both factors are negative. If last is -, factors have opposite signs.
Method 2: Quadratic Formula
x = (-b ± sqrt(b² - 4ac)) / 2a
Use when factorization is not obvious in 5 seconds
Comparison Questions

Two equations given. Compare x and y.

  1. Solve both equations to get x values and y values
  2. Compare: If all x > all y → x > y
  3. If all x < all y → x < y
  4. If x values span across y values → No relation
SHORTCUT: You don't always need exact roots. If equation I gives positive roots and equation II gives negative roots → x > y immediately. Check signs first before calculating.
Example
I. x² - 9x + 20 = 0
II. y² - 13y + 42 = 0
I: x² - 9x + 20 = 0 → (x - 4)(x - 5) = 0 → x = 4, 5 II: y² - 13y + 42 = 0 → (y - 6)(y - 7) = 0 → y = 6, 7 Compare: x = 4 or 5, y = 6 or 7. Both x values < both y values.
Answer: x < y
05

Data Interpretation (DI)

8-10 questions. The biggest chunk. Master percentage change and ratio.
% Increase
(New - Old) / Old x 100
% Decrease
(Old - New) / Old x 100
What % is A of B?
A / B x 100
% More than B
(A - B) / B x 100
% Less than B
(B - A) / B x 100
If A is X% more than B
A = B x (100+X) / 100
If A is X% less than B
A = B x (100-X) / 100
Successive % change
a + b + (a x b)/100
+20% then -20% = -4%
DI Speed Trick

Never calculate exact when approximate works

  1. Bar chart showing 124, 186, 248 → Think in ratios: 124:186:248 ≈ 2:3:4
  2. Pie chart percentages: 35% of total → 0.35 x total. If total = 2000, 35% = 700.
  3. Average = Total / Number of items. For 5 years, sum all and divide by 5.
  4. Ratio questions: A:B = 3:4 means A = 3/7 of total, B = 4/7 of total.
SHORTCUT: If DI asks "Which year had highest growth?", don't calculate all. Visually compare bars/lines first. Often you can eyeball the answer. Calculate only when years are close.
Example — Bar Chart
A company's sales: 2019=200cr, 2020=240cr, 2021=300cr, 2022=270cr. What is % increase from 2019 to 2021?
Increase = 300 - 200 = 100 % increase = 100/200 x 100 = 50%
Answer: 50%
Common Trap: "% increase" vs "times increase." If sales went from 100 to 200: % increase = 100%, but "increased by 2 times" means became 3x (100 + 200). Read carefully.
06

Profit & Loss

2-3 questions. Core formulas. No variation.
Profit
Profit = Selling Price - Cost Price
% Profit = (Profit / CP) x 100
Loss
Loss = Cost Price - Selling Price
% Loss = (Loss / CP) x 100
Marked Price & Discount
SP = MP x (100 - Discount%) / 100
Successive Discounts
Equivalent = a + b - (a x b)/100
20% + 10% = 28% (not 30%)
Example
An article marked at Rs 800 is sold at 2 successive discounts of 20% and 10%. Find selling price.
Method 1: Equivalent discount = 20 + 10 - (20x10)/100 = 30 - 2 = 28% SP = 800 x (100-28)/100 = 800 x 0.72 = 576 Method 2: After 20% discount: 800 x 0.8 = 640. After 10% on 640: 640 x 0.9 = 576
Answer: Rs 576
07

Simple & Compound Interest

2-3 questions. Two formulas. Everything derives from them.
Simple Interest: SI = (P x R x T) / 100
Amount = P + SI = P(1 + RT/100)
Compound Interest: A = P(1 + R/100)^T
CI = A - P = P[(1 + R/100)^T - 1]
Shortcut for Compound Interest

When rates are standard percentages, use multiplier method

  1. 10% for 2 years: multiplier = 1.10 x 1.10 = 1.21. A = P x 1.21. CI = 0.21P
  2. 20% for 2 years: 1.20 x 1.20 = 1.44. CI = 0.44P = 44%
  3. 10% for 3 years: 1.10 x 1.10 x 1.10 = 1.331. CI = 33.1%
MEMORIZE: 10% 2yr = 21%, 10% 3yr = 33.1%, 20% 2yr = 44%. These appear repeatedly.
Example
Rs 10,000 at 10% CI for 2 years. Find amount.
Multiplier = 1.10 x 1.10 = 1.21 Amount = 10,000 x 1.21 = 12,100 CI = 12,100 - 10,000 = 2,100
Answer: Amount = Rs 12,100, CI = Rs 2,100
Half-yearly CI: Rate becomes R/2, Time becomes 2T. Quarterly: Rate = R/4, Time = 4T. Formula same: A = P(1 + R/200)^(2T) for half-yearly.
08

Time & Work

2-3 questions. LCM method is fastest.
The LCM Method

Never use fractions. Use LCM of days.

  1. A can do work in 10 days, B in 15 days. LCM of 10 and 15 = 30.
  2. Total work = 30 units.
  3. A's 1 day work = 30/10 = 3 units. B's 1 day work = 30/15 = 2 units.
  4. Together: 3 + 2 = 5 units/day. Time = 30/5 = 6 days.
TRICK: Take LCM of all given days. That becomes total work. Divide by each person's days to get daily output. Add for combined, subtract for leaving.
Example — Work Leaves
A and B can do a work in 12 and 18 days. A leaves after 4 days. How long will B take to finish?
LCM of 12 and 18 = 36. Total work = 36 units. A's daily = 36/12 = 3 units. B's daily = 36/18 = 2 units. Together for 4 days: (3+2) x 4 = 20 units done. Remaining = 36 - 20 = 16 units. B alone: 16/2 = 8 days.
Answer: 8 days
Efficiency ratio: If A:B = 2:3, then A's time : B's time = 3:2 (inverse ratio)
More efficient = less time. Efficiency and time are inversely proportional.
09

Time, Speed & Distance

2-3 questions. One formula: Distance = Speed x Time.
Basic
D = S x T
S = D/T
T = D/S
Average Speed
Total Distance / Total Time
NOT average of speeds
Same distance, two speeds
Avg Speed = 2ab/(a+b)
60 km/h and 40 km/h → avg = 48 km/h
Relative Speed (same direction)
S1 - S2 (faster - slower)
Relative Speed (opposite)
S1 + S2
Boat & Stream
Downstream = B + S
Upstream = B - S
B = boat speed, S = stream speed
Example
A train 200m long crosses a platform 300m long in 25 seconds. Find speed in km/h.
Total distance = train + platform = 200 + 300 = 500m Speed = 500m / 25s = 20 m/s Convert to km/h: 20 x (18/5) = 72 km/h
Answer: 72 km/h
m/s to km/h: Multiply by 18/5. km/h to m/s: Multiply by 5/18. Memorize: 18/5 = 3.6. So 20 m/s = 20 x 3.6 = 72 km/h.
10

Mixture & Alligation

1-2 questions. Visual method.
Alligation Rule:
(Cheaper quantity) : (Dearer quantity) = (Dearer price - Mean) : (Mean - Cheaper price)
Mean = the price/percentage of the mixture
Example
Mix rice at Rs 20/kg with rice at Rs 30/kg to get mixture at Rs 26/kg. Find ratio.
Mean = 26. Cheaper = 20. Dearer = 30. Cheaper quantity : Dearer quantity = (30 - 26) : (26 - 20) = 4 : 6 = 2 : 3
Answer: 2:3
Replacement formula: If a vessel contains X liters of milk, Y liters removed and replaced with water, repeated n times: Milk left = X(1 - Y/X)^n. This is actually CI formula disguised.
