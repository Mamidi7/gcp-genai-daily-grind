04 Animations

Page Entrance

@keyframes pageIn { from { opacity: 0; transform: translateY(18px); } to { opacity: 1; transform: translateY(0); } }

Every <section> gets this. Trigger via IntersectionObserver adding class "in".

SVG Pipeline Flow

@keyframes pipeFlow { to { stroke-dashoffset: -18; } }

Animated arrow for data flow diagrams. Use with stroke-dasharray: 6 3.

Stamp Entrance

@keyframes stampIn { from { opacity: 0; transform: rotate(-12deg) scale(1.6); } to { opacity: .85; transform: rotate(3deg) scale(1); } }

Hero accent stamp. Typically positioned position: absolute; top: 0; right: 0; and animated with animation: stampIn .4s .6s cubic-bezier(.2,.8,.3,1.4) forwards;

Pulse (for indicators)

@keyframes pulse { 0%,100%{opacity:1;} 50%{opacity:.4;} }

For "live" indicators, loading states.

06 How to Use

1. Copy the CSS variables block

:root { --paper: #f3eee4; --paper-2: #ebe4d4; --ink: #1a1612; --ink-2: #3d3528; --ink-dim: #6b5d49; --ink-faint: #a89a83; --rule: #c8b99e; --accent: #c0392b; --accent2: #a93226; --green: #2d7d46; --blue: #2b6cb0; --gold: #b8860b; --serif: Georgia, 'Times New Roman', serif; --sans: 'Inter', system-ui, sans-serif; --mono: 'IBM Plex Mono', ui-monospace, monospace; --highlight: #fff59d; }

2. Include dark mode override

[data-theme="dark"] { --paper: #1b1917; --paper-2: #242220; --ink: #f0ece9; --ink-2: #c5bbb2; --ink-dim: #9a9088; --ink-faint: #6b625c; --rule: #3a3430; --accent: #e8704a; --accent2: #d95a3a; --green: #5bbd6b; --blue: #5b8fbd; --gold: #d4a84b; --highlight: #7a5a3a; }

3. Add dark toggle JS

document.getElementById('darkToggle').addEventListener('click', function() { var d = document.documentElement; if (d.getAttribute('data-theme') === 'dark') { d.removeAttribute('data-theme'); } else { d.setAttribute('data-theme', 'dark'); } });

4. Import fonts

<link href="https://fonts.googleapis.com/css2? family=IBM+Plex+Mono:wght@400;500;600& family=Inter:wght@300;400;500;600&display=swap" rel="stylesheet">
