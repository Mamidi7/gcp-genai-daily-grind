## Errors Debugged

- **Pydantic V1 vs V2 confusion** — @validator deprecated in V2, replaced by @field_validator + @classmethod pattern. Symptom: code review caught it. Fix: use `@field_validator("field")` with `@classmethod` decorator.
- **Retry decorator target selection** — Which exceptions to catch. Root cause: Google API errors hierarchy. Fix: catch `ResourceExhausted` (quota) and `ServiceUnavailable` (transient), let everything else propagate.
- **Terraform depends_on missing** — Cloud Run resource referenced `google_service_account.ai_workload.email` before the SA was created. Fix: add `depends_on` to Cloud Run resource.
- **Empty jitter range** — Random jitter of `random.uniform(0, 0)` adds no value. Fix: `random.uniform(0, 0.5)` for real jitter to prevent thundering herd on retry.
