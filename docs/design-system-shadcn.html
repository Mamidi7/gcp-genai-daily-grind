Layer Caching

Docker optimization

Copy requirements.txt first, pip install, then copy code. Avoids reinstalling deps on every build.

Port Binding

0.0.0.0 not 127.0.0.1

Inside a container, 127.0.0.1 binds to localhost only. Cloud Run health check won't reach it.

04. How to Use

Copy the CSS block below into any HTML file to get shadcn theming.

/* 1. Paste this in any HTML <style> block */ :root { --background: #ffffff; --foreground: #09090b; --card: #ffffff; --card-foreground: #09090b; --primary: #18181b; --primary-foreground: #fafafa; --secondary: #f4f4f5; --secondary-foreground: #18181b; --muted: #f4f4f5; --muted-foreground: #71717a; --border: #e4e4e7; --input: #e4e4e7; --ring: #18181b; --radius: 0.5rem; --font-sans: 'Inter', sans-serif; --font-mono: 'JetBrains Mono', monospace; } /* 2. Add dark mode support */ @media (prefers-color-scheme: dark) { :root { ... } } /* 3. Load fonts */ <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600&family=JetBrains+Mono:wght@400;500&display=swap" rel="stylesheet"> /* 4. Use classes: sh-btn, sh-card, sh-badge, sh-table, sh-code, sh-tabs */
