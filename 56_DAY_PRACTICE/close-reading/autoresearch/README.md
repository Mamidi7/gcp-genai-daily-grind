## What is Autoresearch?

**Autoresearch** = AI runs ML experiments autonomously while you sleep!

- **Single GPU** - No distributed setup needed
- **5-minute experiments** - Fast iteration cycle
- **~12 experiments/hour** - Run 100+ overnight
- **Self-correcting** - Keeps improvements, discards failures
