# Day 10: Cloud Storage (GCS) Basics

## Mission
Master Google Cloud Storage - buckets, objects, and access control. Essential for storing models, datasets, and artifacts.

## Today's Topic
**Cloud Storage (GCS)**

### Key Concepts
1. **Buckets** - Container for objects (like S3 buckets)
2. **Objects** - Files stored in buckets (any file type)
3. **Storage Classes** - Standard, Nearline, Coldline, Archive
4. **Access Control** - ACLs, IAM, Signed URLs

---

## Why This Matters for Interviews

- "How would you store a trained ML model?"
- "What's the difference between Nearline and Coldline?"
- "How do you give temporary access to a file?"
