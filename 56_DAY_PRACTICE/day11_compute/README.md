# Day 11: Compute Services

## Mission
Understand GCP compute options - when to use each (Compute Engine, Cloud Run, Cloud Functions, GKE).

## Today's Topic
**GCP Compute Services**

### Key Concepts
1. **Compute Engine** - Virtual machines (full control)
2. **Cloud Run** - Containerized apps (serverless containers)
3. **Cloud Functions** - Serverless functions (event-driven)
4. **GKE** - Kubernetes (container orchestration)

---

## When to Use What?

| Service | Use Case | Scaling |
|---------|----------|---------|
| Compute Engine | Full OS control, custom software | Manual |
| Cloud Run | Containerized web apps/API | Auto |
| Cloud Functions | Simple event-driven tasks | Auto |
| GKE | Complex microservices | Auto |
