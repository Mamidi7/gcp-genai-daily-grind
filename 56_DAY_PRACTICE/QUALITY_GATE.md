# Quality Gate

Score each day out of 10.
If score is below 10, the day is not closed.

## 10-Point Checklist

1. Topic explained in simple English
2. ASCII or other clear visual exists
3. Block-wise why is explained
4. Manual-first steps are present
5. One relevant example is used consistently
6. One failure path is documented
7. One code or SQL/BQ artifact exists
8. One interview answer exists
9. Handoff files are updated
10. Green-streak artifact is ready

## Fix Rule

If a point is missing, add the missing material on the same day:
- no visual -> add ASCII
- no example -> add one business example
- no failure -> run or simulate one error path
- no interview answer -> write 30-second version
- no handoff -> update state and closeout

Only then mark the day complete.
