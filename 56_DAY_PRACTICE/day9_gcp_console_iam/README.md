# Day 9: GCP Console + IAM Basics

## Mission
Master GCP Console navigation, project structure, and IAM roles - essential foundations for any GCP job interview.

## Today's Topic
**GCP Console, Projects & IAM Basics**

### Key Concepts
1. **GCP Console** - Web interface for managing all GCP resources
2. **Projects** - Organizational containers for resources
3. **IAM Roles** - Viewer, Editor, Owner, and custom roles

---

## Why This Matters for Interviews

- Interviewers ask: "What's the difference between Viewer and Editor?"
- You MUST know how to navigate Console without mouse
- IAM questions are common: "How would you give someone read-only access?"

---

## Learning Objectives

- [ ] Navigate GCP Console (console.cloud.google.com)
- [ ] Understand project structure and resource hierarchy
- [ ] Know the difference between IAM primitive roles
- [ ] Create a custom role (optional challenge)
