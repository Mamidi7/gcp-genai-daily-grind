## Day 6: Decision Trees 🌳

**Date:** March 2, 2026
**Resource:** [MLU-Explain Decision Trees](https://mlu-explain.github.io/decision-tree/)

### 1-Sentence Summary
A Decision Tree splits data by asking yes/no questions on features, choosing splits that maximize information gain (reduce uncertainty).

### Key Formulas
- **Entropy:** `H(S) = -Σ p_i * log₂(p_i)`
- **Information Gain:** `IG(S, A) = H(S) - Σ (|S_v|/|S|) * H(S_v)`

### Interview Answer Ready?
- [ ] Can explain in 2 min: YES / NO
- [ ] Can answer "How does entropy decide splits?": YES / NO

### Punch Dialogue
> "A decision tree splits data based on features to maximize information gain — asking 'what question separates the most data?' at each step. It's intuitive, interpretable, and serves as the foundation for Random Forests."

## Day 10: Precision & Recall 🎯

**Date:** _________
**Resource:** [MLU-Explain Precision-Recall](https://mlu-explain.github.io/precision-recall/)

### Key Formulas
- **Precision:** `TP / (TP + FP)`
- **Recall:** `TP / (TP + FN)`
- **F1 Score:** `2 * (Precision * Recall) / (Precision + Recall)`
